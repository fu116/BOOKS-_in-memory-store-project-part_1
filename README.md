# Milestone Project 2: Book Tracker Application

A clean, command-line interface (CLI) application built with Python and SQLite to manage and track reading lists. This project marks a transition from flat-file storage to relational database systems.

## 🚀 Features

- **Add New Books:** Store book titles and authors.
- **List Reading Inventory:** View all books alongside their read status.
- **Update Status:** Dynamically mark books as read, instantly reflecting in the database.
- **Delete Entries:** Easily remove titles from your collection using foolproof query handling.

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Database:** SQLite3
- **Tools:** VS Code, Database Client extension
